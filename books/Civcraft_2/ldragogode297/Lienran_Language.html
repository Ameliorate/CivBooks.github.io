Plurals add 'ux' to the end of the name, except for these letters: t, which changes to lox. n, which changes to mox. g, which changes to rux.
Page 11 of 16
and h, rth, and th, which just simply add an x on the end. When a word has two of the same letters next to each other, it is only written as one. Every three Lienran letters a apostrophe is added.
